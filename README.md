# c-programming
A repo to document my journey with learning C and reading [C Programming Language, 2nd Edition](docs/pdf/the-C-programming-language-2nd-edition.pdf)

- [C Snippets in VSCode](https://marketplace.visualstudio.com/items?itemName=Harry-Ross-Software.c-snippets)


| Input     | Description |
| :-------- | :----------- |
| sst       | Starter Template |
| libsst    | Starter template with stlib.h |
| if        | If statement |
| elif      | Else if statement |
| else      | Else statement |
| for       | For loop |
| while     | While loop |
| dowhile   | Do...while loop |
| ig	    | Header file include guard |
| libsst    | Linked list template |
| intfunc	| Create int function |
| flfunc	| Create float function |
| strfunc	| Create string function |
| longfunc	| Create long function |
| vtdef	    | Create definition for virtual table |
| vtfunc	| Create function for virtual table |
| pflo	    | Print variable of type float (2 decimal places) |
| pint	    | Print variable of type int |
| pcha	    | Print variable of type char |
| ppoint    | Print variable of type pointer |
| psiz	    | Print variable of type size_t |
| cal	    | Allocate memory using calloc |
